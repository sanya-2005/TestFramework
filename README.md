# TestFramework
Фреймворк, позволяющий создавать юнит-тесты на языке С++. Работает через загрузку DLL, работает только для Windows.

Смотри x64/Debug/help.txt для пояснения.

После чего положить получившуюся DLL в папку tests рядом с runtests.exe, и запустить этот файл.

Смотри исходники для дополнительной информации.
